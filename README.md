# таджвид.рф — экзамен по таджвиду

Сайт экзамена первого уровня по таджвиду (наука чтения Корана). Преподаватель: Деаб Анас Т.

- **Сайт:** https://таджвид.рф (GitHub Pages, ветка `main`)
- **Бэкенд:** приватный репозиторий `noreal001/tajweed-backend`, Railway; адрес API — в [assets/config.js](assets/config.js)
- **Экзамен в Markdown:** [docs/exam.md](docs/exam.md) · оригинал: [docs/original-exam-level-1.pdf](docs/original-exam-level-1.pdf)

## Как это работает

1. Ученик выбирает: пройти экзамен или оставить заявку на уроки. В заявке можно выбрать несколько дней, диапазон времени и отключаемый звуковой отклик ползунка.
2. Экзамен: 51 письменный вопрос по одному на экран (3 минуты на вопрос, назад нельзя) + запись чтения вслух с микрофона.
3. Ответы уходят на сервер, там считается процент письменной части (ключ ответов только на сервере). Ученик получает постоянную ссылку на результат и секретный личный кабинет с историей сдач.
4. Преподаватель смотрит результаты, расписание заявок и посещаемость в админке бэкенда (`/admin`) или закрытом Telegram-боте.

## DNS на Reg.ru (домен таджвид.рф)

В зоне домена `таджвид.рф` (punycode `xn--80aefbin9d.xn--p1ai`) создать записи:

| Тип | Имя | Значение |
|-----|-----|----------|
| A | @ | 185.199.108.153 |
| A | @ | 185.199.109.153 |
| A | @ | 185.199.110.153 |
| A | @ | 185.199.111.153 |
| AAAA | @ | 2606:50c0:8000::153 |
| AAAA | @ | 2606:50c0:8001::153 |
| AAAA | @ | 2606:50c0:8002::153 |
| AAAA | @ | 2606:50c0:8003::153 |
| CNAME | www | noreal001.github.io. |

Все прочие A/AAAA записи для `@` (например, парковку Reg.ru) удалить. После обновления DNS (от нескольких минут до 24 часов из-за кэшей провайдеров) в настройках репозитория GitHub → Pages появится галочка у домена, тогда включить «Enforce HTTPS».

```bash
ADMIN_PASSWORD=test node server.js
```
